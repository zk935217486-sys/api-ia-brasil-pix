## 1. LinkedIn - post principal

### Título opcional

Como comparar o custo de Claude e Codex no Brasil sem se enganar

### Texto

Um gateway de acesso para API de IA não se prova com um percentual solto.

Para comparar Claude, Codex ou um fluxo ChatGPT-like, eu usaria sempre a mesma tarefa:

- tokens de entrada e saída;
- quantidade de execuções no mês;
- câmbio, IOF e spread se houver cartão internacional;
- custo final em BRL;
- custo por tarefa, não só custo por chamada.

## Ordem de publicação

1. Publicar a calculadora no GitHub README e em uma discussão técnica.
2. Publicar o post do TabNews com a pergunta no final.
3. Publicar o LinkedIn no dia seguinte, apontando para a página de evidências.
4. Responder apenas tópicos que já falem de Claude, Codex, Pix, IOF ou custo de API.
5. Medir no Google Analytics os cliques por origem e não repetir a mesma mensagem em vários grupos.

Não publique os cinco textos no mesmo dia. Links repetidos em comunidades diferentes parecem distribuição automática e reduzem a chance de aprovação.
