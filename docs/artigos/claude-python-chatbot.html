Como criar um chatbot Claude no terminal em Python

Um exemplo pequeno para testar resposta do modelo, custo e latência antes de levar a integração para um produto.

Instale o SDK

pip install anthropic

Configure seu token

export TOKEN_AI_API_KEY="seu_token_aqui"

Código completo

import os
from anthropic import Anthropic

client = Anthropic(api_key=os.environ["TOKEN_AI_API_KEY"])

print("Chat Claude. Digite 'sair' para encerrar.")

while True:
    prompt = input("Voce: ")
    if prompt.lower() == "sair":
        break

    msg = client.messages.create(
        model="claude-sonnet-4-20250514",
        max_tokens=500,
        messages=[{"role": "user", "content": prompt}],
    )
    print("Claude:", msg.content[0].text)
