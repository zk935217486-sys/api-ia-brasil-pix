Codex preco Real Brasil: como calcular sem surpresa na fatura

O custo real de Codex no Brasil quase nunca e so o token. Entra IOF, cambio, tentativa repetida e o tamanho do diff que voce manda para o modelo.

Regra simples de bolso

prs_mes = 24
tentativas_por_pr = 2
tokens_por_tentativa = 9200
preco_por_mil_tokens = 0.0  # ajuste aqui com seu plano

tokens_mes = prs_mes * tentativas_por_pr * tokens_por_tentativa
custo_estimado = (tokens_mes / 1000) * preco_por_mil_tokens
print(tokens_mes, custo_estimado)

O que mexe mais no custo

  • Quantidade de contexto enviado em cada PR.
  • Numero de reprocessamentos por diff.
  • Conversao USD para BRL no fechamento.
  • IOF em cartao internacional.
