Codex é útil para revisão de PR, geração de testes e refactor. O problema é quando o acesso ao token depende de cartão internacional, dólar e burocracia de cobrança.

Por que isso importa?

Em projetos de código, consumo cresce rápido: contexto do repositório, diff grande, várias tentativas e prompts de correção. Se você só quer validar o fluxo, faz sentido começar com pouco saldo e medir.

tentativas_por_pr = 4
tokens_por_tentativa = 9000

tokens_semana = prs_por_semana * tentativas_por_pr * tokens_por_tentativa
print(tokens_semana)
